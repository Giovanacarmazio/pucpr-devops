# EnergyOps Automation

Este repositório contém o projeto **EnergyOps Automation**, focado na automação de processos de gestão de energia utilizando práticas de DevOps. Desenvolvido como parte de estudos na **Pontifícia Universidade Católica do Paraná (PUCPR)**.

## Descrição do Projeto

O projeto demonstra como práticas DevOps podem ser aplicadas em sistemas de automação energética para garantir resiliência e eficiência em ambientes descentralizados de energia.

## Tecnologias Utilizadas

- **Python**
- **GitHub Actions**
- **Docker**
- **Kubernetes**
- **Prometheus e Grafana**

## Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/Giovanacarmazio/PUCPR-Devops.git
   ```

2. Execute o script Python:
   ```bash
   python hello.py
   ```

3. Visualize pipelines no GitHub Actions.
